# متطلب تقني: API تسجيل دخول الأدمن (Admin Login)

يرجى من مطوري الباك اند (Backend) إنشاء API مخصص لتسجيل دخول الأدمن بالمواصفات التالية:

*   **نوع الطلب:** POST
*   **البيانات المطلوبة (Payload):**
    *   `email`: البريد الإلكتروني الخاص بالأدمن.
    *   `password`: كلمة السر.
*   **الهدف:** عند محاولة الدخول على رابط `/admin` في الموقع، سيتم تحويل الأدمن لصفحة تسجيل الدخول، وبمجرد إدخال البيانات، يجب أن تقوم الـ API بالتحقق وإرجاع الـ Token الخاص بالأدمن لفتح لوحة التحكم.

---
رابط لوحة التحكم الحالي: `https://thyro-care-x-6jdn.vercel.app/admin`
